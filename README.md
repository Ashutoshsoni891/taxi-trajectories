# T-Drive Data Challenge 

http://startup.ml/challenge/

In Beijing, Microsoft collected GPS data from 10,357 taxis. 
Sampling each taxi's location every 177 seconds. 


http://research.microsoft.com/apps/pubs/?id=152883
